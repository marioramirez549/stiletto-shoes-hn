# Stiletto Shoes HN

Catálogo web oficial de Stiletto Shoes HN.

## Sitio web

[Visitar el catálogo](https://marioramirez549.github.io/stiletto-shoes-hn/)

## Cómo editar productos

1. Abre [site/catalogo.json](site/catalogo.json).
2. Presiona el ícono del lápiz.
3. Modifica únicamente los datos que están entre comillas.
4. Presiona **Commit changes**.
5. Espera aproximadamente un minuto mientras GitHub publica la actualización.

### Campos disponibles

- `nombre`: nombre que aparecerá en el catálogo.
- `categoria`: Mocasines, Tacones, Destalonados u otra categoría.
- `colores`: colores disponibles.
- `precio`: escribe, por ejemplo, `L 1,250`.
- `tallas`: escribe, por ejemplo, `Tallas 6, 7, 8 y 9`.
- `imagen`: ubicación de la fotografía dentro de `site/images`.
- `etiqueta`: mensaje pequeño sobre la fotografía.
- `destacado`: usa `true` para mostrar ese producto en la portada.
- `disponible`: usa `true` si está disponible o `false` si está agotado.

Para añadir una fotografía, súbela primero a la carpeta `site/images` y luego escribe su ruta en el producto, por ejemplo: `images/nuevo-modelo.jpg`.
